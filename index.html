О HTML

HTML (HyperText Markup Language) — это стандартный язык разметки для создания веб-страниц. HTML описывает структуру веб-страницы и состоит из набора элементов, которые указывают браузеру, как отображать контент.

Этот сайт демонстрирует различные HTML-элементы, стилизованные с помощью CSS.

Основные элементы

HTML включает множество элементов для структурирования контента:

  • Заголовки (h1-h6)
  • Параграфы (p)
  • Ссылки (a)
  • Изображения (img)
  • Списки (ul, ol, li)

Пример кода HTML

Простая HTML-структура:

<!DOCTYPE html>
<html>
<head>
  <title>Заголовок страницы</title>
</head>
<body>
  <h1>Мой заголовок</h1>
  <p>Мой первый параграф.</p>
</body>
</html>

Таблицы в HTML

HTML таблицы полезны для отображения структурированных данных:

Тег Описание
<table> Определяет таблицу
<tr> Определяет строку таблицы
<td> Определяет ячейку таблицы
<th> Определяет заголовок таблицы
